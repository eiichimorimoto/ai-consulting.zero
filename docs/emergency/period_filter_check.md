# 🔍 Google Cloud Console 期間フィルター変更手順

## 問題

30日間のグラフでは131回しかリクエストがない
でも429エラーが出ている

→ **今日のデータだけを見る必要がある**

---

## 📊 期間フィルターの変更手順

### Step 1: 画面上部の期間フィルターを探す

現在の設定:
```
[✓ 30日] ← ここをクリック
```

### Step 2: 期間を変更

選択肢:
```
□ 1時間
□ 6時間
□ 12時間
□ 1日     ← まずこれを選択
□ 2日
□ 4日
□ 7日
□ 14日
□ 30日
```

### Step 3: グラフを確認

**確認すべきこと**:
1. 「レスポンスコード別のトラフィック」グラフ
   - 今日のリクエスト数は？
   - 400エラー（赤）の数は？
   - 200成功（緑）の数は？

2. グラフの縦軸の値
   - 25,000に近い数字が表示されるか？

---

## 🔍 さらに詳しく確認

### 次に確認すべきページ: クォータ（割り当て）

```
URL: https://console.cloud.google.com/apis/api/pagespeedonline.googleapis.com/quotas
```

**このページで確認できること**:
```
Queries per day
現在値: _____ / 25,000

↑ ここに実際の使用量が表示される
```

---

## 📸 確認してほしいスクリーンショット

以下の2つをスクリーンショットで確認してください:

### 1. 期間を「1日」に変更したグラフ
- レスポンスコード別のトラフィック
- 今日のリクエスト数

### 2. クォータページ
- Queries per day の現在値
- 使用率のバー
