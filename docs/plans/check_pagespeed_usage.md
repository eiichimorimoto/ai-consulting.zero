# Google PageSpeed API 使用状況の確認手順

## 📊 確認方法

### 1. Google Cloud Console にアクセス

```
URL: https://console.cloud.google.com/
```

### 2. プロジェクトを選択

- 左上のプロジェクト名をクリック
- PageSpeed APIキーを作成したプロジェクトを選択

### 3. APIダッシュボードを開く

```
左メニュー
  ↓
「APIとサービス」
  ↓
「ダッシュボード」
```

### 4. PageSpeed Insights API を探す

- API一覧から「PageSpeed Insights API」をクリック
- または検索ボックスに「PageSpeed」と入力

### 5. 指標タブで確認

**確認できる情報：**
- ✅ リクエスト数（過去24時間）
- ✅ エラー数
- ✅ レイテンシ
- ✅ 時間別の使用状況グラフ

---

## 🔍 クォータの確認

### 1. 割り当てページを開く

```
左メニュー
  ↓
「APIとサービス」
  ↓
「割り当て」
```

### 2. PageSpeed API を検索

- 検索ボックスに「PageSpeed」と入力

### 3. 確認できる情報

```
項目                   値
────────────────────────
1日あたりのリクエスト   25,000
現在の使用量           ???? / 25,000
リセット時刻           太平洋時間 00:00
```

---

## 💡 今日の使用量を確認したら

### 使用量が多い場合（20,000超え）
- 今日は分析機能の使用を控える
- 17:00（日本時間）のリセットを待つ
- Supabaseキャッシュで済む範囲でテスト

### 使用量が少ない場合（10,000未満）
- キャッシュが効いている証拠
- 通常通り使用可能
